# Portfolio Project – 100Hires

This project demonstrates my ability to follow structured instructions, learn new tools independently, and clearly document my process using GitHub and AI-assisted development tools.
---
## Tools Installed
- Cursor IDE (AI-powered code editor)
- Claude Code extension (installed via Cursor Extensions)
- Codex extension (installed via Cursor Extensions)
- GitHub account for version control and project sharing
---
## Steps Completed
1. Installed Cursor IDE from the official website
2. Opened Cursor and navigated to the Extensions marketplace
3. Installed both Claude Code and Codex extensions and logged in
4. Created a GitHub account
5. Created a new public repository named `Project100hires`
6. Opened the repository folder in Cursor IDE
7. Created this `README.md` file to document the process
8. Committed changes locally using Git
9. Pushed the repository to GitHub  
---
## Challenges & Solutions
**Issue 1: Unable to find extensions initially**  
- I first searched in the browser instead of inside Cursor
- **Solution:** Navigated to the Extensions tab within Cursor and searched there
   
**Issue 2: Confusion about Git commands (commit vs push)**
- I was unfamiliar with how changes are saved and uploaded
- **Solution:** Watched beginner tutorials and learned:
- *Commit* = saving changes locally
- *Push* = uploading changes to GitHub
   
**Issue 3: Setting up repository correctly**
- Initially unsure how to connect local folder to GitHub
- **Solution:** Followed step-by-step tutorials and retried until successful  
---
## What I Learned
- How to use GitHub to create and manage repositories
- The difference between commit and push in version control
- How to install and use extensions in Cursor IDE
- How to troubleshoot technical issues independently using online resources
- The importance of clear documentation in projects

This task showed me that even simple instructions require attention to detail and persistence. I improved my confidence in using GitHub and learned how to solve problems independently. Documenting each step also helped me organize my thinking more clearly.
