# Setup & Configuration Project

## Overview
This project documents the setup of development tools and basic GitHub workflow.

## Tools Installed
- Cursor IDE
- Claude Code extension
- Codex extension
- Git
- GitHub
  
## Steps Completed
1) Installed Cursor IDE
2) Installed Claude Code extension and logged in
3) Installed Codex extension and logged in
4) Created a GitHub account  
5) Created a public repository  
6) Opened the repository in Cursor  
7) Created a README.md file  
8) Initialized Git and pushed the project  

## Issues Encountered

**Issue 1:** Git not recognized in terminal  
**Solution:** Installed Git and restarted the terminal  

**Issue 2:** Failed to push to GitHub  
**Solution:** Used Personal Access Token instead of password
